Desafio Final

Agora, que você já tem conhecimento o bastante de Docker vamos fazer uma atividade final, essa atividade é um exemplo do mundo real com uma empresa real:
A SWIFT está avaliando a necessidade de construção de um novo portal institucional para apresentação e consolidação da marca junto aos clientes e ao mercado de proteínas. Porém é necessário construir um ambiente de homologação (também conhecido como ambiente de validação), para que as alterações realizadas pela equipe de TI e Designer esteja disponível neste novo ambiente sem interferir com o processo que existe atualmente.

Pense nisso, você faz parte da equipe que deverá atender essa nova demande de atividade formando um SQUAD Ágil com diversos perfis e conhecimentos em TI. Deste modo, é necessário prover um ambiente em CONTAINER conforme imagem abaixo: Para desenvolvimento desta atividade, é necessário atender as seguintes demandas:

Database Server

  1. O banco de dados deve utilizar a última versão disponível do POSTGRESQL:latest - É necessário possibilitar o acesso ao banco de dados através da rede utilizando a porta 5454
  2. Você deve definir a senha de acesso formada por seu <nome_idade>
  3. Você deve carregar os scripts no banco dados disponíveis no arquivo 06a - Lab DockerCompose_Atividade.zip
Web Server
  1. O servidor web utilizado pelo banco original é o NODE.JS na versão 14 ou superior.
  2. Você deve carregar uma aplicação temporária disponível no arquivo 06d - LabDockerComposeAtividade_Node.zip para a pasta do container / <seunome-app>
  3. O servidor web em NODE.JS deverá possibilitar acesso do host através da porta 8080
Projeto Swift
  1. Utilizar o arquivo 06d – LabDockerComposeAtividade_swift.zip para carregar no Web Server definido no item anterior
  2. Deverá ser configurado um VOLUME que permita compartilhar a pasta no container com o notebook local, de modo que uma alteração no arquivo HTML já reflita automaticamente no container sem a necessidade de reconstrução da imagem ou do container

Regras para construção da arquitetura da ATIVIDADE FINAL, você deve ter as seguintes informações:

  1. No mínimo um arquivo DOCKERFILE
  2. Um arquivo DOCKER-COMPOSE

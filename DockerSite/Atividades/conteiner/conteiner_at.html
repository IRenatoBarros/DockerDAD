1ª questão

Qual comando você usaria para acessar o terminal de um contêiner em execução chamado "ub"?

docker connect -it ub bash docker attach -it ub bash docker exec -it ub bash docker open -it ub bash

2ª questão

Qual comando é usado para mapear a porta 80 do host para a porta 8080 de um contêiner?

docker run -p 8080:80 app docker run -p 80:8080 app docker run --port 8080:80 app docker run --port 80:8080 app
